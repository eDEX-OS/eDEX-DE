eDEX-DE

A fullscreen sci-fi desktop environment frontend for Hyprland — built in Rust + TypeScript/Preact with Tauri v2.

⬇ Install Guide GitHub →

Features

Everything you need for a sci-fi Hyprland DE

eDEX-DE replaces your desktop with a fully-functional, tron-themed terminal environment with deep Linux integration.

🖥️

Terminal DE

Fullscreen multi-tab terminal built on xterm.js + Rust PTY server. Up to 5 concurrent sessions with WebSocket transport.

🪟

Hyprland IPC

Live window list, workspace switcher, active window title, and a built-in config generator with window rules and keybinds.

🚀

App Launcher

Spotlight-style fuzzy launcher over .desktop apps. Open with Alt+Space or Meta. Supports shell commands and math eval.

📦

Native Packages

.deb for Debian/Ubuntu, .rpm for Fedora, PKGBUILD for Arch AUR. Built and published automatically on version tags via GitHub Actions.

🔌

System Integrations

PipeWire/PulseAudio volume control, NetworkManager WiFi panel, fprintd fingerprint auth, systemd service manager with live logs.

🎨

Sci-fi Themes

Tron-inspired default theme with full CSS custom property theming. Customize colors, fonts, and glow effects via settings.

Architecture

How it works

Tauri v2 bridges a Rust backend and a TypeScript/Preact frontend over a type-safe IPC layer, with direct Unix socket connections to Hyprland.

Install

Get eDEX-DE on your system

🟠 Debian / Ubuntu

bash
wget https://github.com/eDEX-OS/eDEX-DE/releases/latest/download/edex-de_amd64.deb
sudo dpkg -i edex-de_amd64.deb

🔵 Fedora

bash
sudo rpm -i https://github.com/eDEX-OS/eDEX-DE/releases/latest/download/edex-de.x86_64.rpm

🐧 Arch Linux

bash
yay -S edex-de
# or: paru -S edex-de

Press Alt+Space from within eDEX-DE for the Spotlight-style launcher. For the Meta (Super) key, add bind = SUPER, Space, exec, edex-de --launcher to your hyprland.conf.
eDEX-DE supports PipeWire (via wpctl) and PulseAudio (via pactl). It automatically detects which is available. If neither is found, the volume control is hidden gracefully.
