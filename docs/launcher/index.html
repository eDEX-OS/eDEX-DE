Launcher

App Launcher Guide

The launcher gives eDEX-DE a Spotlight-style command surface for apps, shell commands, and quick math. It is optimized for keyboard-first workflows.

Hotkeys

Alt+Space or Meta+Space

Alt+Space is handled inside eDEX-DE and opens the launcher overlay instantly. Meta+Space needs a Hyprland bind because Wayland apps cannot reserve the Super key globally.

ShortcutScopeBehavior
Alt+SpaceBuilt-inOpens the launcher overlay from anywhere inside the running eDEX-DE window.
SUPER+SpaceHyprland bindLaunches edex-de --launcher or toggles a running instance when configured externally.

Discovery

.desktop file locations scanned

The launcher searches common XDG application paths so it can index both system packages and per-user app installs.

LocationTypical contents
/usr/share/applicationsSystem-installed desktop files from distro packages.
/usr/local/share/applicationsManually installed or locally built application entries.
~/.local/share/applicationsUser-specific desktop files and overrides.
~/.local/share/flatpak/exports/share/applicationsFlatpak app exports for the current user.

Hyprland bind

Meta+Space setup

Add this line to hyprland.conf if you want the launcher on the Super key:

hyprland.conf
bind = SUPER, Space, exec, edex-de --launcher
