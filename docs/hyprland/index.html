Hyprland

Hyprland Integration Guide

eDEX-DE talks directly to Hyprland over its Unix sockets to read workspaces, dispatch commands, and stream live events without an extra compatibility layer.

IPC

Socket overview

Hyprland exposes two sockets per running compositor instance. eDEX-DE reads the active instance signature from the environment and connects directly.

SocketPathUsed for
.socket.sock$XDG_RUNTIME_DIR/hypr/$HYPRLAND_INSTANCE_SIGNATURE/.socket.sockRequest/response commands such as workspace queries and dispatch actions.
.socket2.sock$XDG_RUNTIME_DIR/hypr/$HYPRLAND_INSTANCE_SIGNATURE/.socket2.sockStreaming event feed for windows, active workspace changes, and focus updates.

Capabilities

Window manager features

  • Live workspace and active window status in the UI
  • One-click workspace switching via Hyprland dispatch commands
  • Built-in config generator for rules, startup, and keybinds
  • Graceful fallback when eDEX-DE is launched outside Hyprland

Generator

Using the Hyprland Config Generator

  1. Open eDEX-DE and press Ctrl+Shift+H.
  2. Review the generated rules, launcher bind, and startup block.
  3. Click save to write ~/.config/edex-de/hyprland-integration.conf.
  4. Add a source = ~/.config/edex-de/hyprland-integration.conf line to your main hyprland.conf.

Snippets

Recommended hyprland.conf blocks

Startup
exec-once = edex-de
Launcher bind
bind = SUPER, Space, exec, edex-de --launcher
Window rules
windowrulev2 = pin, class:^(edex-de)$
windowrulev2 = nodecor, class:^(edex-de)$
windowrulev2 = noborder, class:^(edex-de)$
windowrulev2 = noshadow, class:^(edex-de)$
windowrulev2 = fullscreenstate, 1, class:^(edex-de)$
windowrulev2 = noanim, class:^(edex-de)$

Events

Hyprland events reference

EventMeaningTypical UI use
workspaceUser switched active workspace.Refresh the workspace bar highlight and counts.
activewindowFocused window changed.Update title display and active app metadata.
openwindowA new client appeared.Refresh the window list and occupancy indicators.
closewindowA client exited.Remove stale entries from the live window list.
movewindowA client moved workspaces or monitors.Update placement state without polling.
