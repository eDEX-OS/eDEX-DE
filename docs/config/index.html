Settings

Settings JSON reference

Current builds store JSON keys in camelCase. The table includes the common labels used in docs alongside their on-disk key names where helpful.
KeyTypeDefaultDescription
themestringtronControls the active color palette. Unknown values fall back to the default tron-inspired theme.
nointrobooleanfalseSkip the animated boot intro and jump directly into the main UI.
cwdstring~Initial working directory for terminals and the file browser panel.
shellstringbashShell executable launched by the PTY backend, such as bash, zsh, or fish.
termFontSize (font_size)number15Terminal font size used by xterm.js. Increase it for high-DPI monitors or decrease for dense dashboards.
shellOpacity (opacity)number1.0Terminal pane opacity from fully opaque 1.0 down to translucent values such as 0.85.

Themes

Built-in and custom themes

eDEX-DE ships with a neon-first default theme set and applies colors through CSS custom properties so the whole interface updates consistently.

  • tron — the primary cyan/green eDEX-DE look, matching the docs site and default UI.
  • default — a more neutral black-and-white preset that keeps the same layout without the heavy glow effect.
  • Custom themes — advanced users can patch the theme map in the frontend and expose new palette names through settings.
settings.json example
{
  "theme": "tron",
  "nointro": false,
  "shell": "bash",
  "cwd": "~",
  "termFontSize": 15,
  "shellOpacity": 1.0
}

Files

Config file locations

On Linux, Tauri stores app configuration under the application config directory. In practice, that gives you a per-user settings area plus the Hyprland helper file generated by eDEX-DE.

FileTypical locationPurpose
settings.json~/.config/<app-id>/settings.jsonPersistent UI, shell, theme, and startup settings.
shortcuts.json~/.config/<app-id>/shortcuts.jsonUser-overridable keyboard shortcuts and automation mappings.
hyprland-integration.conf~/.config/edex-de/hyprland-integration.confGenerated Hyprland rules and binds you can source from hyprland.conf.
