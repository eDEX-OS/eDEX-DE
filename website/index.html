eDEX-DE

A sci-fi Wayland desktop environment — built in pure Rust

GitHub Latest Release

// FEATURES

🖥️

Full Wayland Compositor

Built on smithay — DRM/KMS, libinput, XDG-shell. No Hyprland, no X11 required.

🎨

eDEX-UI Faithful Aesthetic

3-panel sci-fi layout, glowing cyan borders, animated hex keyboard. Tron, Matrix, and Amber themes.

💻

Built-in Terminal

PTY-backed terminal emulator with multi-tab support, ANSI/VT100 rendering, and clipboard.

📊

System Dashboard

Live CPU, RAM, network, disk, and process stats rendered directly in the UI.

🚀

App Launcher

Alt+Space fuzzy app launcher scanning XDG .desktop files across all standard paths.

🪟

Tiling Window Manager

Smart tiling: 0 apps → eDEX fullscreen; 1 app → fullscreen; 2+ → equal columns. No escape from eDEX.

⚙️

Settings Panel

14-category settings UI — appearance, display, input, audio, network, security, terminal, and more.

🔒

Privacy Indicators

Live status bar indicators for Tailscale, WireGuard, Tor, fprintd, microphone, and camera.

// INSTALLATION

Arch Linux (AUR)

yay -S edex-de

Then log out and select eDEX-DE from your display manager.

Debian / Ubuntu

sudo dpkg -i edex-de-*-amd64.deb
Download .deb

Fedora / RHEL

sudo rpm -i edex-de-*.x86_64.rpm
Download .rpm

From Source

cargo build --release -p edex-de
sudo install -Dm755 \
  target/release/edex-de \
  /usr/bin/edex-de
